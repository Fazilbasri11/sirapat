❓ FAQ — Cara Penggunaan

1. Kenapa data arsip hilang saat clear browser? ▼
Data arsip kini disimpan permanen di Google Sheets via GAS. localStorage hanya sebagai cache cepat. Pastikan URL GAS sudah diisi di Pengaturan — data akan tersedia di browser manapun.
2. Cara deploy Google Apps Script? ▼
  1. Buka script.google.com → Proyek Baru
  2. Paste isi file gas_backend_fixed.js
  3. Isi SS_SURAT_ID, SS_ARSIP_ID, dan DRIVE_FOLDER_ID
  4. Deploy → Web App → Execute as: Me · Access: Anyone
  5. Salin URL → paste ke Pengaturan SiRapat
3. Cara generate dokumen rapat? ▼
  1. Pilih tanggal, jam, tempat, agenda, dan centang peserta
  2. Klik Generate 3 Dokumen — ZIP otomatis terunduh dan arsip langsung disimpan ke cloud
  3. Cek di menu Arsip — data sudah tersinkron
