💻 Tentang DocuMeet

Nama Sistem
DocuMeet — Sistem Rapat Digital KIP Pidie Jaya
Fungsi Utama
Generate otomatis 3 dokumen rapat dari template Word
Teknologi
HTML5 · docxtemplater · Google Apps Script · Drive
Penyimpanan
Google Sheets (permanen) + localStorage (cache)

❓ FAQ

1. Kenapa data arsip hilang saat clear browser? ▼
Data arsip disimpan permanen di Google Sheets via GAS. localStorage hanya cache. Pastikan URL GAS diisi di Pengaturan.
2. Cara generate dokumen rapat? ▼
  1. Pilih tanggal, jam, tempat, agenda, centang peserta
  2. Klik Generate 3 Dokumen — file .docx terunduh otomatis
  3. Arsip langsung tersimpan ke cloud
